# LifeLine GPS — Agent simulation in a graph

JavaFX application used to visualize ambulance movement in a weighted graph. Ambulances move from node to node by following a path computed by a shortest path algorithm (Dijkstra, A*, or Congestion-Aware).

## Installation and Launch

### Command Line

1. Clone the repository:
```bash
   git clone https://github.com/bGlmZW/Projet-Info-CY-GI5A ProjetGenieLogiciel
```

2. Next, type these commands to compile and run the application

**Linux / macOS**

```bash
# Compile
javac --module-path /path/to/javafx-sdk-21/lib \
      --add-modules javafx.controls,javafx.graphics \
      -d bin \
      $(find src -name "*.java")

# Run
java --module-path /path/to/javafx-sdk-21/lib \
     --add-modules javafx.controls,javafx.graphics \
     -cp bin view.MainApp
```

**Windows PowerShell**

```powershell
# Compile
javac --module-path "C:\path\to\javafx-sdk-21\lib" `
      --add-modules javafx.controls,javafx.graphics `
      -d bin `
      (Get-ChildItem -Recurse -Filter "*.java" src).FullName

# Run
java --module-path "C:\path\to\javafx-sdk-21\lib" `
     --add-modules javafx.controls,javafx.graphics `
     -cp bin view.MainApp
```

Replace `/path/to/javafx-sdk-21` with the actual JavaFX SDK path on your machine.

### With Eclipse

1. Clone the repository:
```bash
   git clone https://github.com/bGlmZW/Projet-Info-CY-GI5A ProjetGenieLogiciel
```

2. In Eclipse: **File → New → Java Project**
   - Uncheck **"Use default location"**
   - In Location, select the cloned folder
   - Click **Next**, then **Finish**

3. Configure JavaFX in the Build Path:
   - Right-click the project → **Build Path → Configure Build Path**
   - Tab **Libraries → Modulepath → Add External JARs**
   - Select all `.jar` inside the `lib/` folder of the downloaded JavaFX SDK
   - **Apply and Close**

4. Update the `src/module-info.java` file. Eclipse generates an empty one when creating the project. Replace its content with:
```java
   module <project_name> {
   	requires transitive javafx.controls;
        requires transitive javafx.graphics;

    	exports model.accident;
    	exports model.agent;
    	exports model.graph;
    	exports pathfinding;
    	exports simulation;
    	exports controller;
    	exports view;
    	exports io;
   }
```
Replace `<project_name>` with your project name generated by Eclipse.

5. Run `MainApp.java` located in `src/view/` with **Run As → Java Application**

## Usage

When the application starts, a demonstration graph is displayed. The toolbar at the top is used to edit the graph, while the control bar at the bottom controls the simulation.

### Graph Editing

| Button | Action |
|---|---|
| Add Node | Click on an empty area to create a node |
| Add Edge | Click two nodes to create an edge |
| Add Ambulance | Select a node, then enter the destination, speed, and algorithm |
| Random Expansion | Add random nodes and edges to expand the graph |
| Random Agents | Add several randomly generated agents |
| Edit Selection | Edit the selected node, edge, or agent |
| Delete Selection | Delete the selected node, edge, or agent |
| Clear All | Remove all nodes, edges, and agents |
| Save | Save the graph state as a binary file |
| Load | Load the graph state from a binary file |

### Simulation Controls

| Button | Action |
|---|---|
| Start | Run the simulation continuously |
| Pause | Pause the simulation |
| Next Tick | Advance the simulation manually by one tick |
| Reset | Reset the simulation |
| Speed Slider | Change the delay between ticks |
| Arrival Behavior | Choose what happens when an agent reaches its destination |
| Pathfinding Algorithm | Choose the default pathfinding algorithm |
| Help | Open the user manual |

### Available Algorithms

- **Dijkstra** — computes the shortest path in a weighted graph.
- **A\*** — uses a heuristic to speed up pathfinding.
- **Congestion-Aware** — takes traffic congestion into account when computing paths.

## Accident and Patient Information

When an accident node is selected:

- Accident information is displayed.
- Patient data is displayed in the patient panel.
- BPM and temperature are updated at each simulation tick.
